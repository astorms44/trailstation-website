# trailstation-website

Public landing, privacy policy, and support pages for the **Trailstation** iPhone + Apple Watch app, hosted on GitHub Pages.

## URLs
- Landing: https://astorms44.github.io/trailstation-website/
- Privacy Policy: https://astorms44.github.io/trailstation-website/privacy.html
- Support: https://astorms44.github.io/trailstation-website/support.html

## Files
- `index.html` — landing page
- `privacy.html` — privacy policy (used in App Store Connect → App Privacy → Privacy Policy URL)
- `support.html` — support page (used in App Store Connect → Support URL)
- `style.css` — shared styles (light/dark)

## To update
```bash
cd "/Users/austinstorms/Documents/Custom Software/Trailstation/trailstation-website"
# edit files
git add . && git commit -m "..." && git push
```
GitHub Pages rebuilds in about a minute.
